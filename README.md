# Sistema de Vendas e Estoque — SQL

Projeto de modelagem e implementação de banco de dados relacional utilizando SQL e MySQL Workbench.

## Objetivo

Desenvolver um banco de dados para gerenciamento de:

* clientes
* produtos
* vendas
* colaboradores
* categorias
* estoque

## Modelagens Desenvolvidas

* Modelo Conceitual
* Modelo Lógico
* Modelo Físico

## Tecnologias Utilizadas

* SQL
* MySQL
* MySQL Workbench

## Funcionalidades do Banco

* Cadastro de clientes
* Controle de produtos
* Registro de vendas
* Relacionamento entre tabelas
* Controle de categorias
* Integridade referencial
